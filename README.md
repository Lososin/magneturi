### Install:  
____________
```
$ go get github.com/nmmh/magneturi/magneturi 
$ cd %GO_PATH%/src/github.com/nmmh/magneturi/magneturi

$ go build; go install
-or-
$ go test
```

Import 
____________
```
import "github.com/nmmh/magneturi/magneturi"
```
